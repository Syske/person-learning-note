# electron-forge安装及配置

#### 全局安装

```sh
npm install -g @electron-forge/cli
```

#### 项目中安装

```
npm install --save-dev @electron-forge/cli
```



#### 初始化项目



```
electron-forge init electron-forge-demo
```
