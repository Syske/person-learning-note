#### 知识点汇总

- 基本类型可以通过==，或者！=进行条件判断
- Byte是byte的包装类型，初始化为null而不是0;
- intValue()是把Integer对象类型变成int的基础数据类型； 
- parseInt()是把String 变成int的基础数据类型； 
- Valueof()是把String转化成Integer对象类型；（现在JDK版本支持自动装箱拆箱了。）
