# vim使用技巧

#### 替换

```
:%s/要替换内容/替换后内容[\g]
```

`\g`如果省略表示替换当前行的数据，否则表示替换所有

#### 启用鼠标复制粘贴

```
:set mouse=
```

#### 设置文件编码
```
:set ff=unix
```

#### 文件编码转换
```
:set fileencoding=utf-8
```

#### 显示行号
```
:set nu
```

#### 隐藏行号
```
:set nonu
```

### 统计单词出现次数

```
# 先搜索关键词，例如搜索 fail
:/fail
# 然后通过下面的命令，统计出现次数
:%s///gn
```
