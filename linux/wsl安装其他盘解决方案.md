# wsl安装其他盘解决方案

最近有小伙伴在问`wsl`默认安装在`C`盘的问题，觉得安装在`C`盘太占内存，毕竟一般新买的电脑默认`C`盘也就分了`100`多`G`，时间久了确实不够用。之前我还真没关注过这个问题，但是最近我发现我的`C`盘也越来越小了，甚至这里几天已经飘红了
