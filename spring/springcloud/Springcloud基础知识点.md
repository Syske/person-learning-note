# Spring cloud基础知识点

### 常用模块

springCloud集成了如下模块：

- 配置管理（configuration management）
- 服务发现（service discovery）
- 断路器（circuit breakers）
- 智能路由（intelligent routing）
- 微代理（micro-proxy）
- 控制总线（control bus）
- 一次性令牌（one-time tokens）
- 全局锁（global locks）
- 分布式选主（leadership election）
- 分布式会话（distributed sessions）
- 群集状态（cluster state）
