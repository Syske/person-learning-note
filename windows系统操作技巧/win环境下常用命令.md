# win环境下常用命令

### 跟踪网络

跟踪网络跳跃点

```
tracert 127.0.0.1
```

![](https://gitee.com/sysker/picBed/raw/master/images/20210526105045.png)



#### 查询系统信息

```
wmic bios
```

![](https://gitee.com/sysker/picBed/raw/master/images/20211102141619.png)

比如查询序列号：

```
wmic bios get SerialNumber
```
