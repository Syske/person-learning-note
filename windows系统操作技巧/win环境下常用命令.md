# win环境下常用命令

### 跟踪网络

跟踪网络跳跃点

```
tracert 127.0.0.1
```

#### 查询系统信息

```
wmic bios
```

![](
https://syske-pic-bed.oss-cn-hangzhou.aliyuncs.com/imgs/images/20211102141619.png)

比如查询序列号：

```
wmic bios get SerialNumber
```
